# Macro CLKS

![image](https://github.com/viniimiguel/autoclick/assets/144070822/e26a402f-a53a-4b59-9848-7f81c68a1c50)

### O Macro Clks é uma ferramenta poderosa que atualmente desempenha funções essenciais, incluindo autoclick e hotkeys. Com esta aplicação, você tem a capacidade de configurar a execução de várias teclas com apenas um comando, proporcionando praticidade e eficiência.

# Recursos Principais

## Auto Click:

### O Macro Clks oferece a funcionalidade de autoclick, automatizando cliques repetitivos e otimizando tarefas que exigem interações frequentes do mouse.

![image](https://github.com/viniimiguel/autoclick/assets/144070822/33830c9e-be35-4ee0-82f2-5ed6497f4e5a)


![image](https://github.com/viniimiguel/autoclick/assets/144070822/05beba07-e324-4e57-97b5-cb7dd0acf856)


### Após segurar a tecla (F7) o macro vai começar a reconhecer seus clicks tanto com o botão esquerdo quanto com o botão direito.

![image](https://github.com/viniimiguel/autoclick/assets/144070822/bea1aeb5-b99b-4560-895e-48bc078170ae)
